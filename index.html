Problem: Subarray with Maximum Sum

Given an array of integers (positive, negative, or zero), find the contiguous subarray with the maximum sum and return its sum.

Function Signature:

int maxSubArraySum(vector<int>& nums);

Sample Input:

nums = [-2, 1, -3, 4, -1, 2, 1, -5, 4]

Sample Output:

6

Explanation: The subarray [4, -1, 2, 1] has the largest sum = 6.

Edge Case Input:

nums = [-5, -4, -3, -2, -1]

Output:

-1
